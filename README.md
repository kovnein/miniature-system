# miniature-system
NZ Spelling Bee · Practice Hive

Small web app created to help practice for NZ spelling bee 2026

This project was built with the help of an AI assistant (Claude, by Anthropic) — including code, word definitions, and the phonetic pronunciation guides used for Te Reo Māori words.
 
A few things worth knowing if you're using or contributing to this:
 
- **General word definitions and example sentences** were AI-generated and spot-checked, but not independently verified against a dictionary for every single entry. If you spot an error, please open an issue or PR.
- **Te Reo Māori phonetic pronunciations** (the respelled text fed to the text-to-speech engine, e.g. `tikanga` → `tee-kahng-ah`) are an approximation designed to coax a browser's English text-to-speech voice closer to correct Māori pronunciation. They are **not sourced from or verified by a fluent speaker or an authoritative source like [Te Aka Māori Dictionary](https://maoridictionary.co.nz)**, and dialectal pronunciation varies regionally in ways this simplified approach can't capture. Treat them as a rough guide, not a definitive reference — for actual pronunciation guidance, Te Aka's own audio recordings are the better source.
- **Code** (the app logic, UI, and word-list structure) was written collaboratively with AI assistance and has been tested manually, but hasn't been through a formal code review process.
If you fork or reuse this project, you're welcome to correct, improve, or replace any of the above — contributions that improve accuracy (especially around Te Reo pronunciation) are very welcome.
